Tutorial

GPyS acts as an interface to your GPX data. In this short tutorial, we'll be passing a GPX file to GPyS and performing basic GIS operations on its data.

Set up

First, we'll need to import GPyS into our project.

import GPyS

We will primarily interact with our GPS data via the GPXFile object.

Say that we have a file called marathon.gpx. All we need to do is pass it to the GPXFile constructor.

myMarathon = GPXFile(marathon.gpx)

We can then call functions on myMarathon and it will return the appropriate values.

Interfacing with our data

Say that we want to know the total distance travelled during the marathon. There is a handy class method called .totalTrackLength that tells us just that.

myMarathon.totalTrackLength() # This returns 42197 meters!

How about the timestamp when we were halfway through the marathon?

n = myMarathon.totaltrkpts() / 2
median_trkpt = myMarathon.trkptNumber(n)
median_trkpt.timestamp() # Returns our desired timestamp!

In fact, those last two lines could be condensed into a single line like so.

median_trkpt_time = myMarathon.trkptNumber(n).timestamp() # This works fine too.

Imagine that Konbiniya was one of the rest stations on the track and we wanted to know how far that was from the halfway point.

We pull the coordinates from a quick Google search and add it as a TrackPoint object.

rest_station = TrackPoint(-123.1279, 49.2864)
median_trkpt.distanceFrom(rest_station) # Returns 322 meters!

With this new TrackPoint, .distanceFrom() returns the distance along the track from that rest station to our halfway point.

And we're done!
