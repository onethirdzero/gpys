Tutorial

GPyS acts as an interface to your GPX data. In this short tutorial, we'll be passing a GPX file to GPyS and performing basic GIS operations on its data.

Set up

First, we'll need to import GPyS into our project. We can do this the same way as with a regular Python module.

import GPyS

Next, we'll need to instantiate a GPXFile object to contain the GPX file data.

Say that we have a file called marathon.gpx. We pass it to GPyS using the GPXFile constructor like so.

myMarathon = GPXFile(marathon.gpx)

We can then treat myMarathon as a representation of marathon.gpx's data. That is, we can call functions on it and expect it to return the appropriate values.

Interfacing with our data

Say that we want to know the total distance travelled during the marathon. We can call the class method .totalTrackDistance on myMarathon.

myMarathon.totalTrackDistance() # This returns 18372 meters!

How about the timestamp when we were halfway through the marathon?

n = myMarathon.totaltrkpts() / 2
median_trkpt = myMarathon.trkptNumber(n)

We first extracted the median track point using .totaltrkpts() and some simple arithmetic and stored it in a variable called n. Then we used n as an argument to a class method called .trkptNumber(), which returns the n-th TrackPoint object in the myMarathon.

Finally, we call .timestamp on median_trkpt.

median_trkpt.timestamp() # Returns our desired timestamp!

And we're done!

For more comprehensive information on each class and its methods, refer to the Reference Doc.
